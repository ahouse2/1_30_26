## Operational Sequence (Recommended)
1. Configure providers and credentials (`settings_and_providers.md`).
2. Ingest case files (`ingestion.md`).
3. Validate graph and timeline outputs (`graph_explorer.md`, `timeline_builder.md`).
4. Run forensics and presentation packaging (`forensics_suite.md`, `presentation_studio.md`).
5. Execute research/strategy and trial prep (`research_strategy.md`, `trial_prep_hub.md`).
6. Run voice and orchestration stations as needed (`voice_assistant.md`, `centcom_war_room.md`).
7. Track closure tasks in parity console (`parity_ops.md`).
